1. Общие положения

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей Telegram-бота «Горздрав Бот» (далее — «Сервис»).

Используя Сервис, вы даёте согласие на обработку ваших персональных данных в соответствии с настоящей Политикой.

Термины и определения

  • Оператор — владелец и администратор Сервиса, осуществляющий обработку персональных данных.
  • Персональные данные (ПД) — любая информация, относящаяся к прямо или косвенно определённому физическому лицу.
  • Обработка ПД — любое действие с ПД, включая сбор, запись, хранение, использование, передачу.
  • Субъект ПД — пользователь Сервиса, чьи персональные данные обрабатываются.

2. Собираемые персональные данные

При использовании Сервиса мы собираем следующие категории данных:

Идентификационные данные

  • Telegram User ID
  • Имя пользователя в Telegram (@username)
  • Имя и фамилия (если указаны в профиле)

Медицинские данные

  • Текстовые описания симптомов
  • Результаты анализа медицинских изображений (временная обработка)
  • Информация о записях к врачу (район, поликлиника, время)

Технические данные

  • Время и дата взаимодействий с ботом
  • Выбранные функции и команды
  • Анонимизированные метрики использования

Важно: Медицинские изображения (фото анализов) обрабатываются временно и не хранятся после получения результата анализа.

Данные об использовании

  • История запросов и ответов бота
  • Частота использования функций
  • Ошибки и технические проблемы

Данные, которые НЕ собираются

  • Номер телефона пользователя
  • Финансовая информация
  • Локация в реальном времени
  • Контакты из телефонной книги

4. Правовые основания обработки

Основные правовые основания:

1

Согласие субъекта ПД

Вы даёте явное согласие на обработку персональных данных при первом использовании бота через команду /start.

2

Исполнение договора

Обработка необходима для предоставления услуг, запрошенных пользователем (запись к врачу, анализ симптомов).

3

Законные интересы оператора

Обработка для улучшения качества сервиса и обеспечения его безопасности.

4

Общедоступные данные

Имя пользователя Telegram является общедоступной информацией.

6. Права субъектов персональных данных

В соответствии с законодательством РФ, вы имеете право:

Право на доступ

Получить информацию об обработке ваших персональных данных

Право на исправление

Исправить неточные или неполные персональные данные

Право на удаление

Удалить ваши персональные данные (право на забвение)

Право на отзыв согласия

Отозвать согласие на обработку персональных данных

Право на ограничение

Ограничить обработку в определённых случаях

Право на переносимость

Получить данные в структурированном формате

Как реализовать ваши права:

Для реализации ваших прав отправьте команду в Telegram-боте:

/privacy_info — Получить информацию об обработке ваших данных
/delete_data — Удалить все ваши персональные данные
/export_data — Экспортировать ваши данные в формате JSON
