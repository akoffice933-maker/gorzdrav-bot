Как отменить запись к врачу?

Отменить запись можно несколькими способами:

Через бота (рекомендуется)

  1. Откройте бот в Telegram
  2. Отправьте команду /cancel
  3. Выберите запись для отмены из списка
  4. Подтвердите отмену

Через меню бота

  1. Нажмите на кнопку меню (три линии)
  2. Выберите "Мои записи"
  3. Найдите нужную запись
  4. Нажмите "Отменить запись"

Прямое обращение

  1. Позвоните в вашу поликлинику
  2. Сообщите данные о записи
  3. Попросите отменить приём

Важная информация:

  • Отмена доступна не позднее чем за 24 часа до приёма
  • При частых отменах система может ограничить возможность записи
  • Освободившееся время становится доступно другим пациентам

Безопасно ли отправлять фото анализов?

Безопасно и конфиденциально

Все данные обрабатываются с соблюдением требований 152-ФЗ

Шифрование данных

Передача и хранение данных осуществляется с использованием шифрования

Медицинские изображения не хранятся после получения результата

Совет: Перед отправкой убедитесь, что на фото нет другой личной информации кроме результатов анализов.

Какие анализы можно расшифровать?

Система поддерживает расшифровку большинства распространённых анализов:

Кровь

  • Общий анализ крови
  • Биохимический анализ
  • Коагулограмма
  • Гормональные исследования

Моча

  • Общий анализ мочи
  • Анализ по Нечипоренко
  • Суточный анализ

Прочие

  • Копрограмма
  • Мазки
  • Справки с результатами

Ограничения:

  • Не поддерживаются: рентгеновские снимки, КТ, МРТ
  • Требуется чёткое фото с хорошим освещением
  • Текст должен быть читаемым и на русском языке

Важное предупреждение:

Результаты расшифровки носят информационный характер и не являются медицинским диагнозом.

Всегда консультируйтесь с врачом для постановки точного диагноза и назначения лечения.

Факторы, влияющие на точность:

  • Качество фотографии
  • Чёткость текста на бланке
  • Стандартность формата анализа
  • Наличие референсных значений

Что такое ЕСИА и зачем авторизация через Госуслуги?

ЕСИА

Единая система идентификации и аутентификации (ЕСИА) — это государственная система, используемая для авторизации на портале Госуслуги и других государственных сервисах.

Зачем нужна авторизация через ЕСИА:

Официальная идентификация

Подтверждение личности пользователя

Требования API Горздрава

Необходимо для работы с официальным API

Безопасность данных

Государственный уровень защиты информации

Процесс авторизации:

  1. Вы выбираете запись к врачу в боте
  2. Бот перенаправляет вас на страницу Госуслуг
  3. Вы входите в свой аккаунт Госуслуг
  4. Система возвращает вас в бота с токеном авторизации
  5. Бот использует токен для работы с API Горздрава

Как удалить свои данные из системы?

Вы можете полностью удалить свои данные несколькими способами:

Через бота (самый простой)

Отправьте команду /delete_data в боте и подтвердите удаление.

Данные удаляются в течение 24 часов

По запросу на email

Отправьте запрос на адрес privacy@gorzdrav-bot.ru

Включите в письмо ваш Telegram ID или username

Через GitHub Issues

Создайте issue в репозитории проекта с запросом на удаление данных

Что именно удаляется:

  • Telegram ID и профиль
  • История записей к врачу
  • Тексты запросов и симптомов
  • Анонимизированные метрики использования

Примечание: Некоторые данные (например, логи для безопасности) могут сохраняться дольше в соответствии с законодательством.
