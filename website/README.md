# Горздрав Бот - Веб-сайт

Веб-сайт для промоушена Telegram-бота "Горздрав Бот", который предоставляет возможность записи к врачу в Санкт-Петербурге через портал Горздрав.

## 📋 О проекте

Этот сайт представляет собой промо-лендинг для Telegram-бота с полным описанием функциональности, FAQ и политикой конфиденциальности.

### Основные страницы:

1. **Главная страница** (`index.html`) - Лендинг с основными возможностями
2. **Возможности** (`features.html`) - Подробное описание всех функций бота
3. **FAQ** (`faq.html`) - Часто задаваемые вопросы с поиском
4. **Политика конфиденциальности** (`privacy.html`) - Документ о защите персональных данных

## 🚀 Установка и запуск

Проект не требует сборки, можно запустить на любом веб-сервере:

### Вариант 1: Локальный сервер Python

```bash
# Перейти в папку проекта
cd website

# Запустить сервер Python
python -m http.server 8000

# Открыть в браузере
open http://localhost:8000
```

### Вариант 2: Node.js (http-server)

```bash
# Установить http-server глобально
npm install -g http-server

# Запустить сервер
http-server website -p 8080

# Открыть в браузере
open http://localhost:8080
```

### Вариант 3: Статический хостинг

Просто загрузите все файлы на любой статический хостинг:
- GitHub Pages
- Netlify
- Vercel
- Обычный веб-сервер (Apache, nginx)
